As a college student studying cybersecurity, I have the great pleasure of taking multiple programming classes, and this semester I happen to be taking Data Structures and Algorithms. I currently finished the second-to-last program, and we went over what the last program would be today, so I feel like I have liberty to discuss this class.

I will try to briefly teach some of the things I learned in this class. Maybe someone will see this and benefit from it, but mainly it helps me retain what I learned. They say teaching is the best way to learn. Here is what I will cover:

  • Pointers and pointer arithmetic

  • Recursion
  • Sorting
  • Searching algorithms
  • Linked lists
  • Writing clean code


Pointers


Pointers are only in select few programming languages, such as C and C++. Pointers are used to manipulate the data of a program at the low levels, enabling the ability to get the best performance and nick-picky you need. But mainly they are used to refer to the same space in memory from multiple locations.
