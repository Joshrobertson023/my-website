Pointers


Pointers are only in select few programming languages, such as C and C++. Pointers are used to manipulate the data of a program at the low levels, enabling the ability to get the best performance and nick-picky you need. But most people will use them to refer to the same space in memory from multiple locations.

Pointers are defined with the * character:


*pointer;

If I wanted a pointer to a variable a:


int var_a = 5;
int *p_var_a = &var_a;

In this case, the memory address of var_a would be stored in p_var_a. Note the use of the address operator (&), to specify to the compiler that we are storing the address location in the pointer.



Side note: it is good practice to name any pointers starting with "p_", that way you don't accidentally write over another value.



You can grab the value of the variable the pointer is pointer to by dereferencing the variable. The dereference operator is the * character. The * character is used to declare pointers, but once a pointer is already declared, the * is then used to dereferece that pointer.


Take a look at this program that prints the value of a variable using a pointer:


int total_students = 32;
int *p_total_students = &total_students;
printf("%d", *p_total_students);

/* --> 32 */

If we omitted the * in the printf statement, we would be printing the memory address of the variable being pointed to, which essentially looks like junk.


int total_students = 32;
int *p_total_students = &total_students;
printf("%d", p_total_students);

/* --> 5673829768342 */


Pointer Arithmetic


Pointer arithmetic is used to move the pointers. If you have an array, pointer arithmetic is how you go up and down the array. Pointer arithmetic can be used on virtually all data structures in C.


To increment a pointer by one, you would use the post-increment operator:


*p_a = &a;
p_a++

Pointers are great to use with arrays. They are more efficient than using indexes (more on indexes later). Say we want an array of student id numbers, and we need to use pointers instead of indexes. First we would declare an array of student ids, then we would declare a pointer:


int student_ids[4] = {65, 22, 06, 49};
int *p_student_id = &student_ids;

That pointer is now pointing to the first element of the array, which is 65. Let's change the value of the first and second elements:


*p_student_id = 99;
*(p_student_id + 1) = 55;

We dereferenced the pointer, to access the value of the element. In the second line, we incremented the pointer before dereferencing it. When we incremented the pointer, it pointed to the next element in the array. The array with the changes made would now be 99, 55, 06, 49.


We can use a for loop, incrementing the pointer to print the entire array:


/* First we need to get the size of the array */
int size = sizeof(student_ids) / sizeof(p_student_id);

/* We also need to declare a moving pointer to go up the array */
int *p_moving_student_id;

/* Now we can loop */
for(p_moving_student_id = p_student_id; p_moving_student_id < size; p_moving_student_id++)
   printf("%d ", *p_moving_student_id);

/* --> 99 55 06 49 */

Notice in the for loop how we increment the moving pointer after every iteration. That is pointer arithmetic in action.
